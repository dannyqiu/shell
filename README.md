StD
---

A shell built by Stanley Lok and Danny Qiu for Mr. DW's 8th Period Systems Class

## Features:

- Colorful prompt
- Home directory recognition: `~`
- Character escapes: `"` & `\` & `'`
- Multiple commands: `;`
- File redirection: `>`
- Handler: `SIGINT`
- Readline library features: Tab Completion & History
